**Agent 1C — KB intelligent** (task #89)

**Agent 1D — Sessions power** (task #90)

- Nouveau modèle `ChatFolder` : id, name, color, order
- UI rail : tri sessions (pinned en haut, archived masquées), filtres folders
- Estimate cost sous bouton Send : `~$X.XX` calculé via tokensIn estimés (longueur message × 1.3)
