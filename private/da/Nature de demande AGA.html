Nature de demande AGA                                                           mise à jour le 09/12/19

 

Ø Réglementation 

Chômage / allocation de garde : AR 26/03/2003

 

Ø Résumé 

-      L’allocation de garde est une indemnité perçue par une gardienne d’enfant suite à l’absence

d’un enfant inscrit chez elle, pour des raisons indépendantes de la volonté de la gardienne.

-      La gardienne d’enfant ne peut avoir ni le statut salarié ni celui d’indépendant ;

elle est simplement liée sans contrat de travail à un service reconnu par la Communauté.

-      Aucun stage d’attente n’est prévu mais l’allocation est octroyée à certaines conditions soit :

=> ne pas avoir d’autre activité (ni salariée, ni indépendante)

=> ne plus être couvert par des indemnités de licenciement

=> ne pas percevoir un autre revenu de remplacement

        (chômage, maladie, repos de maternité, pension complète)

=> ne pas bénéficier d’une indemnité pour accident de travail ou maladie professionnelle

      survenue lors d’une occupation précédente (s’il s’agit d’une incapacité temporaire)

-      Cumul possible avec :

=> journées couvertes par un pécule de vacances

=> activité accessoire en application de l’art. 48 à C1A

=> mandat politique ou public en application de l’art. 49 à C1A

=> activité artistique en application de l’art. 48bis à C1 artiste

=> pension de survie : cumulable pendant 12 mois à C1B

=> allocation de transition

=> allocation d’interruption (crédit-temps ou interruption de carrière)

=> bénévolat à pas de déclaration obligatoire

=> indemnité accident de travail ou maladie professionnelle survenue

     lors d’une occupation précédente à s’il s’agit d’une incapacité permanente à C1B

 

Ø Formulaire à introduire

=> C220A  (N° ONSS du service d’accueil doit être complété)

=> pas de C1 obligatoire (sauf en cas de transfert)

 

Ø Formulaire de paiement 

=> C220B

      (le montant des allocations peut être limité si le partenaire est au chômage,

       afin que celui-ci puisse encore être indemnisé en tant que chef de ménage)

 

Ø Méthode de travail informatisée 

 

1er) écran « sélection demande » (AC040/01) : faire F6 pour création

=> date de demande : date à laquelle la gardienne souhaite obtenir les allocations de garde

=> nature demande : AGA

=> type chômage : 27

=> délai d’introduction : dernier jour du 4ème mois suivant le mois de la demande

 

2ème) écran « signalétique membre » (ML010/03)

=> création de la fiche signalétique

 

3ème) écran « demande – paramètres » (AE035/03)

=> compléter émargement N ou Y pour flux indépendant, alloc. interruption, mutuelle, etc.

=> zone article : à blanc car pas d’admissibilité à vérifier  

 

4ème) écran « demande documents externes écran simplifié » (AE110/06)

=> normalement, pas de flux électronique à réclamer sauf si Y sur écran précédent

 

5ème) écran « liste des émargements » (AE010/01)  

=> encoder l’activité (date début + sélectionner le type de formulaire C220A

                                  + code octroi AGDRO + activité « AGA » + N° ONSS)

           

6ème) écran « demande d’allocations » (AG010/03)

=> état de la demande : valeur 1 (si complet ou F4 sur la zone)

=> zone C2 : Y

=> codification :

à type : 27

=> ajouter commentaires éventuels

 

7ème) écran « modification codification – C2 » (CS010/05)

=> méthode de calcul 08 affichée automatiquement car pas concerné par dégressivité

=> barème (1 seule ligne) s’affiche automatiquement : pas de F20 à faire

 

8ème) écran « signalétique cotisations » (MB010/03)

=> faire affiliation syndicale si pas encore faite + encoder données de la centrale

     ainsi que du mode de paiement (voir formalité selon votre régionale).
