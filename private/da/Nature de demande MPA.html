Nature de demande MPA                                                                       => mise à jour le 25/11/2019

 

Ø Résumé

Demande utilisée lorsqu’un chômeur actif désire modifier uniquement le mode de paiement de ses allocations.

=> !!! soyez attentifs au moment où vous modifier le compte ;

à vérifiez qu’aucun paiement ne soit en cours de traitement.

à si tel est le cas :

- soit, postposer la date du changement de compte après la date de débit

(provoquer l’impression du C9 à la date du changement = code W) 

- soit, annuler le paiement en cours si toujours possible

et ensuite refaire le paiement sur le bon compte

=> remarques :

* si paiement fait sur ancien compte et ce compte est clôturée

à attendre retour paiement avant de repayer (nouveau paiement fait par la compta)

* si chèque circulaire :

- !!! frais pour envoi

- !! en cas de perte: demander à l'intéressé de se rendre dans un bureau BNP-fortis                         avec le numéro de chèque afin d’invalider ce dernier.

=> si la modification est assortie avec un changement d’adresse ou de situation familiale

ou avec l’introduction d’autres documents (C4, C32,…..), vous devez utiliser la demande

s’y rapportant (n’oubliez pas de vérifier la date du dernier paiement effectué)

 

Ø Formulaire à introduire

C1

 

Ø Méthode de travail  informatisée

 

1er) écran « sélection demande » (AC040/01) : faire F6 pour création 

=> date de demande : date du jour de la présentation (!!! sauf si paiement en cours) 

=> nature de demande : MPA

=> type chômage : type de chômage actif pour le membre (01, 03, …)

=> délai d’introduction : fin du mois qui suit le mois de la modification

 

2ème) écran « signalétique du membre » (ML 010/04)

=>  modifier le compte

- si compte belge : tapez uniquement les 12 numéros de compte

- si compte européen (mesure SEPA) : encoder structure complète

(code IBAN – n° de compte – code BIC)

à confirmer l’encodage une 2ème x afin que le programme l’accepte

- si chèque circulaire : encoder valeur 990

=> s’il s’agit d’un numéro de compte : le chômeur doit être titulaire du compte

(ou le titulaire doit faire partie de la situation familiale)

 

3ème) écran « demande – paramètres » (AE035/03)

=> ne pas demander les flux électroniques (mettre N) car demande MPA

=> encoder article 042

 

4ème) écran « composition familiale » (AF010/04)

=> F15 pour imprimer C1

 

5ème) écran « demande d’allocations » (AG010/03)

=> état de la demande : valeur 1 (si complet ou F4 sur la zone)

=> zone « C2 » : Y

=> pas de codification                                             

=> commentaire s’affiche automatiquement

 

Remarque :

=> ne pas oublier de faire également la modification pour la centrale syndicale
