Nature de demande PS                                                              => mise à jour le 17/01/2020

 

Ø Réglementation

Art. 114 § 4, après 2°

 

Ø Résumé

Le montant journalier "forfaitaire" (ainsi que pour la phase 21 – 22 – 23 – 24) d'un chômeur

est augmenté (jusqu'au code BFS) lorsque l'allocation de son partenaire ou conjoint(e)

est indemnisé maximum au code B2A

=> droit au forfait majoré uniquement si les 2 chômeurs sont au chômage complet

     pour tout le mois.

Le chômeur qui demande le forfait majoré : statut cohabitant privilégié

            => droit au code BFS si le chômeur est dans une des phases suivantes : 21 – 22 – 23 – 24

                  ou F à condition que le montant de l’allocation soit moins élevé que le montant BFS

                         

Ø Formulaire de demande

Aucun car pas d’introduction à l’ONEM

 

Ø Formulaire de paiement

C105 à joindre à la carte de contrôle

 

Ø Méthode de travail informatisée

 

1er) écran « sélection demande » (AC040/01) : faire F6 pour création

=> date de demande : toujours au 1er du mois

 car les 2 chômeurs doivent être en chômage complet tout le mois

=> nature demande : PS

=> type chômage : 01, 03, 11,…

=> délai d’introduction : pas de délai (sauf prescription paiement : 3 ans)

 

2ème) écran « composition familiale/C1 » (AF010/01)

=> vérifier que le partenaire soit au chômage (!!! ne peut pas être en 1ère période)

 

3ème) écran « demande d’allocations » (AG010/03)

=> état de la demande : valeur 0 (pas d’intro à l’ONEM)

=> zone C2 : X + date création

=> codification F21 : type chômage + code B

=> zone « 131o » => encodé valeur S

=> ajouter commentaires éventuels    

 

4ème ) écran « modification codification – C2 » p.1

=> méthode de calcul et autres données en fonction du dossier

     (voir fiches help méthodes de calcul)

=> F20 pour calculer barème

=> code BFS ne s’affiche pas 

 

 

Remarques :

 

* au moment du paiement : mettre S dans le calendrier C105

                                            afin que le montant majoré soit indemnisé

 

* pour supprimer ce code S, il faut créer à nouveau une demande PS

   et enlever la valeur S dans la zone C131o
