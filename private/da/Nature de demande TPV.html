Nature de demande TPV                                                           => mise à jour le 10/12/2019

 

Ø Réglementation

Art. 33, art. 32, art. 37 et art. 38 de l’AR du 25/11/1991

 

Ø Résumé

Demande utilisée lorsqu’un travailleur demande pour la 1ère fois les allocations

sur base de ses prestations de travail à temps réduit volontaire (article 33).

=> il est donc toujours important de déterminer le statut du travailleur.

à le chômeur touche des ½ allocations de chômage en fonction de son Q/S  (Q X 12 / S)

 

L’intéressé doit en principe avoir presté au minimum

            * en moyenne 12h/semaine

            ou

            * 1/3 temps si l’horaire tps plein est < 36 heures

 

Le travailleur doit prouver un certain nombre de ½ jours de travail (art. 37) ou assimilés (art. 38) dans une période de référence en fonction de son âge au moment de la demande d’allocations.

=> cette période de référence peut être prolongée par certains événements

(indépendant, détention, éducation d’un enfant,…)

pour autant qu’il y ait au moins un jour dans la période de référence.  

Nombre de ½ jours requis :

Il existe 2 autres possibilités pour être admis sur base de ses prestations de travail :

=> soit, en utilisant la catégorie d’âge supérieur

=> soit, utiliser la carrière professionnelle (article 32) : autorisé à partir de 36 ans 

à c.à.d. qu’il est possible d’utiliser les prestations se trouvant

dans les 10 ans qui précèdent la période de référence =

- soit, prouver ≥ la ½ du nombre de jours requis dans la période de référence initiale

+ prouver au moins 1560 jours dans les 10 ans 

(!!! si carrière plus petite que 10 ans, prouver la ½ du nombre de jours de travail

entre le début de la carrière et la fin de la période de 10 ans)

- soit, prouver ≥ les 2/3 du nombre de jours requis dans la période de référence initiale

+ prouver 8 X le nombre de jours manquants dans les 10 ans

 

Cette nature de demande est également utilisée :

* pour une demande de transfert après une interruption de plus de 28 jours calendrier

* pour une 1ère demande avec type chômage 04

(chômage temporaire pour travail à temps réduit volontaire).

à si l’intéressé n’est pas encore connu dans la régionale

 

Ø Formulaire à introduire

C1 + C4 + annexe éventuellement + DMFA obligatoire (+ autres documents si nécessaire)

 

Ø Formulaire de paiement :

C3A ou autre selon les cas + IDE service placement (sauf si dispensé)

 

Ø Méthode de travail informatisée

 

1er) écran « sélection demande » (AC040/01) : faire F6 pour création

=> date de demande : date à laquelle le chômeur ouvre son droit aux allocations

=> nature demande : TPV

=> type chômage : 03 (ou 04) ou 58 si AGR

=> délai d’introduction : 2 mois de date à date sauf pour type 04 (car CT)

 

 

 

2ème) écran « signalétique membre »  (ML 010/03)

=> créer membre et vérifier messages si ok pour intégration (ONEM et INTEROP)

à si pas relancer via F9 et voir anomalie (F13 pour voir codes et dates intégrations).

=> si transfert = n’oublier pas d’encoder la lettre pour code transfert + C8 transfert   

 

3ème) écran « demande de paramètres » (AE035/03)

=> vérifier le dernier jour indemnisé en chômage complet

(éventuellement, à contrôler dans une autre régionale)

=> compléter Y ou N concernant la demande de documents électroniques

(flux indépendant, flux interruption de carrière, C6, etc.)

=> compléter l’article (F4) : 033

=> contrôler la période de référence (et la modifier si nécessaire)

 

4ème) écran « documents externes » (AE110/06)

=> le programme demande automatiquement la DMFA

+ éventuellement d’autres flux en fonction des Y ou N dans l’écran précédent

=> si besoin de flux électroniques non réclamés par le programme 

è F6 pour demande manuelle

 

5ème ) écran « composition familiale – REGIS (AF030/01)

=> comparaison des données avec celles du REGIS                                    

(adresse, nationalité et composition familiale)

à les données doivent être identiques

à en cas discordance maintenue : encodage d’un code pour justifier la différence

à impression automatique du C1 annexe REGIS à la fin du dossier

 

6ème) écran « composition familiale » (AF010/05)

=> F15 pour imprimer C1

 

7ème) écran « liste des émargements » (AE010/01)

=> encoder le passé professionnel (F6)

à ceci est important pour le C104 par exemple

=> encoder données pour trimestres manquants DMFA si nécessaire (via annexe)

=> utiliser option 6 pour calculer barème

 

8ème) écran « demande d’allocations » (AG010/03)

=> F6 pour accéder à la Time Line è contrôler le résultat

à si des modifications s’avèrent nécessaires (ex. : modification de la P.R),

il faut retourner à l’écran « demande de paramètres » pour y adapter les données.

à ensuite, retour à l’écran « demande d’allocations »

et F6 pour générer une nouvelle Time Line.

=> état de la demande : valeur 1 (si complet ou F4 sur la zone)

=> zone C2 : Y

=> code chiffré : automatique via option 6 sur salaire ou F20

=> zone « type » = 033

à reprend l’article qui a été encodé dans l’écran « demande paramètres »

=> mettre un BL en cas de situation douteuse ou litigieuse

=> ajouter commentaires éventuels    

 

9ème ) écran « modification – codification C2 » p.1

=> nouveau droit = méthode de calcul 01 complétée automatiquement

suite encodage zone type 033 dans écran demande d’allocations

=> F20 pour calculer barèmes 

 

10ème) écran « modification codification – C2 » : page 2 via F8

=> adapter le compteur vacances et le facteur J s’il y a lieu

(effectuer d’abord l’étape 12 si vacances pas encore calculées)

 

11ème) écran « création article 103 » (CS013/03)

=> encoder régime des ½ allocations via Q X 12 / S (attention pour arrondi)

12ème) écran « calcul vacances » (AC200/01)          

=> calculer le facteur J si pas encore fait è F20 sur l’écran

(attention d’avoir l’émargement complet sans zone en surbrillance)

 

13ème) écran « signalétique cotisations » (MB010/03)

=> faire affiliation syndicale si pas encore faite + encoder données de la centrale

ainsi que du mode de paiement (voir formalité selon votre régionale).
